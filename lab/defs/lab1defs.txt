mkdir - make directory ( ex mkdir test )
cd - Change directory ( ex cd /var/www/ )
ls - list files and folders comma separated.
ls -l ( list files with permissions )
ls -la ( list files and hidden files )
pwd ( current directory )

pipes: redirecter output fra script/programmer. feks ls -l | grep .txt sender output fra ls til grep som sorterer.
grep - filtrerer output
cat - printer innholdet fra filer
less - printer innhold fra filer, men kun et skjermbuffer om gangen.
